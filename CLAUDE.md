# MakeGrowth Web Project

## Project Overview
MakeGrowth (메이크그로스) - 이커머스 셀러를 위한 AI 자동화 파트너 웹사이트.
지식창업자가 운영하는 교육+시스템 구축 사업. WHY 중심 설득 구조.

## Tech Stack
- Next.js 16 (App Router, React 19)
- Tailwind CSS v4 (@theme inline, @utility)
- framer-motion 12.x (animations)
- @iconify/react + @iconify-json/solar (icons)
- Outfit (display font) + Pretendard Variable (Korean body)

## Design System
- 디자인 기준: joshua.site (영구 레퍼런스)
- 디자인 토큰: `DESIGN.md` (Pure Monochrome)
- 웹사이트 스펙: `WEBSITE_SPEC.md`
- 사업계획서: `사업계획서_v2.md` (카피 원천)
- 아카이브 (원본 MVP): `docs/archive/`

## Design Direction
- 순수 모노크롬 (#1A1A1A / #666 / #999 / #E0E0E0 / #FFFFFF)
- 보라색 완전 금지
- 타이포그래피 중심 미니멀, 목업/장식 없음
- 전체 화이트 기조, 다크는 CTA/pivot 블록에만
- Supanova easing: cubic-bezier(0.16, 1, 0.3, 1)

## Copywriting Philosophy
- WHY 중심 설득 구조 (기능 나열 X, 공감 스토리텔링 O)
- 페인포인트 → 공감 → 솔루션 → 증거 흐름
- 사업계획서(_v2.md)가 카피의 원천 소스

## Homepage Flow
Hero → MarketReality → KillerSNS → KillerDetailPage → Process → FounderStory → FAQ → FinalCTA

## Killer Features
1. SNS 콘텐츠 자동화 (운영 중)
2. AI 상세페이지 자동화 (설계 완료)

## Conventions
- 한국어 퍼스트 타이포그래피 (`word-break: keep-all`)
- Iconify Solar 아이콘 사용 (Lucide 아님)
- 모션만 Supanova 퀄리티, 비주얼은 joshua 미니멀 우선
