# MakeGrowth Web Project

## Project Overview
MakeGrowth (메이크그로스) - 이커머스 셀러를 위한 AI Transformation 솔루션 랜딩페이지.

## Tech Stack
- Next.js 16 (App Router, React 19)
- Tailwind CSS v4 (@theme inline, @utility)
- framer-motion 12.x (animations)
- @iconify/react + @iconify-json/solar (icons)
- Outfit (display font) + Pretendard Variable (Korean body)

## Design Direction
- Warm Stone (#FAF9F7) + Muted Violet (#8B7CF6) 팔레트
- 히어로만 다크, 나머지 전부 Warm 톤
- Double-Bezel 카드, Floating glass pill 네비게이션
- Supanova easing: cubic-bezier(0.16, 1, 0.3, 1)

## Conventions
- 한국어 퍼스트 타이포그래피 (`word-break: keep-all`)
- Iconify Solar 아이콘 사용 (Lucide 아님)
- Supanova 스킬 기준 최고 퀄리티 적용
